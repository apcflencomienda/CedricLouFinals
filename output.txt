2. Project Overview 
Project Title: Lumos: The AI Enviroment Companion 
The Elevator Pitch: 
Lumos is a portable, AI-driven workspace companion powered by the Arduino R4 WiFi. Unlike 
static sensor kits, Lumos adapts to its environment. Through a 'Chat with Room' interface, the 
user can tell the system its new location (e.), allowing Google Gemini AI to recalibrate its 
analysis of noise, light, and temperature to suit that specific context 
Problem Statement: 
Standard IoT devices use fixed thresholds (e.g., 'If Light < 300, turn on LED'). This fails when 
the device moves; 300 lumens is 'bright' for a bedroom but 'dim' for a library. We need 
Generative AI to understand these changing contexts. By chatting with the device to update 
its location, the system can dynamically shift its 'comfort' definitions without requiring code 
changes.
